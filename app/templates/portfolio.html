System Architecture & Modules

TenshiGuard is not just a tool; it's an ecosystem. Here are the core components I built:

Intelligent Endpoint Agent

A lightweight, Python/PowerShell-based agent that runs on client devices. It captures system telemetry (CPU/RAM), monitors file system events, and enforces security policies in real-time.

Python PowerShell System API

AI Correlation Engine

The brain of the operation. It ingests thousands of events, correlates disparate signals (e.g., failed logins + file download), and uses Machine Learning to detect zero-day anomalies.

Scikit-Learn Heuristics Data Pipelines

Command Center Dashboard

A responsive, real-time admin interface built with Flask and Vanilla JS. Features live charts, remote device management, and an interactive threat map.
